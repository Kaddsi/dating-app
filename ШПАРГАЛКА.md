# 📝 Шпаргалка - Быстрый старт

## 🎯 Что это такое?

Я создал для вас **полноценное приложение для знакомств** в Telegram, которое:
- ❌ **Блокирует** пользователей из России, Беларуси, Ирана, Ирака
- ✅ **Пропускает** всех остальных
- 💜 Красивый дизайн в стиле "матовое стекло"
- 📱 Работает прямо в Telegram (Mini App)
- 💘 Свайпы как в Tinder

---

## 📁 Главные файлы

### Python (Backend)
1. **phone_verification.py** - Проверяет номер телефона и блокирует регионы
2. **main.py** - Telegram бот
3. **swipe_handler.py** - API для свайпов и мэтчей

### React (Frontend)  
4. **Discovery.jsx** - Экран со свайпами
5. **telegram.js** - Подключение к Telegram
6. **App.jsx** - Запуск приложения

### База данных
7. **schema.sql** - Структура PostgreSQL

---

## 🚀 Как запустить (кратко)

### 1. База данных
```bash
sudo apt install postgresql postgresql-postgis
createdb dating_db
psql dating_db < schema.sql
```

### 2. Backend (бот)
```bash
cd bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Создайте файл .env:
# BOT_TOKEN=ваш_токен
# DATABASE_URL=postgresql://...

python main.py
```

### 3. Frontend
```bash
cd frontend
npm install
npm run dev
```

### 4. Telegram бот
- Найдите @BotFather
- Отправьте `/newbot`
- Скопируйте токен в .env
- Настройте Web App URL

## 🔐 Как работает блокировка

```python
# Список запрещённых кодов стран
BLACKLISTED = ["+7", "+375", "+98", "+964"]

# Когда пользователь делится номером:
if номер.startswith("+7"):
    if это_Россия:  # проверка по коду города
        ❌ БЛОКИРОВАТЬ
    else:
        ✅ Казахстан - пропустить
elif номер.startswith("+375", "+98", "+964"):
    ❌ БЛОКИРОВАТЬ
else:
    ✅ ПРОПУСТИТЬ
```
