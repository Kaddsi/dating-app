## ⚡ Быстрый запуск за 5 минут

### Вариант 1: Посмотреть демо (без установки)

1. Откройте файл **`premium-dating-app.html`** в браузере
2. Играйтесь со свайпами, навигацией, кнопками!

### Вариант 2: Запустить локально

**Шаг 1: Установить PostgreSQL**
```bash
# Ubuntu/Debian
sudo apt install postgresql postgresql-postgis

# macOS
brew install postgresql postgis

# Создать базу
createdb dating_db
psql dating_db < database/schema.sql
```

**Шаг 2: Запустить бота**
```bash
cd bot
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# Создать .env файл
echo "BOT_TOKEN=ваш_токен_от_BotFather" > .env
echo "DATABASE_URL=postgresql://localhost/dating_db" >> .env

python main.py
```

**Шаг 3: Запустить фронтенд**
```bash
cd frontend
npm install
npm run dev
```

Откроется на `http://localhost:5173`

## ❓ FAQ

**В: Где взять Telegram Bot Token?**
О: Напишите @BotFather в Telegram → /newbot → следуйте инструкциям

**В: Как изменить список блокируемых стран?**
О: Отредактируйте `BLACKLISTED_PREFIXES` в `phone_verification.py`

**В: Работает ли это без сервера?**
О: Демо (`premium-dating-app.html`) работает без сервера. Полная версия требует backend.

**В: Можно ли добавить свои функции?**
О: Да! Код полностью открыт и готов к расширению.
