# 📂 Структура проекта Premium Dating TWA

```
premium-dating-twa/
│
├── 📖 Документация
│   ├── README.md                    # Основное описание (English)
│   ├── РУКОВОДСТВО.md               # Полное руководство (Русский)
│   ├── ШПАРГАЛКА.md                 # Краткая справка
│   ├── БЫСТРЫЙ_СТАРТ.md             # Инструкция для начинающих
│   ├── DEPLOYMENT.md                # Гайд по деплою
│   └── DELIVERABLES.md              # Что входит в проект
│
├── 🎨 Демо (можно открыть сразу)
│   └── premium-dating-app.html      # Полное работающее приложение
│
├── ⚙️ Конфигурация
│   ├── .gitignore                   # Игнорируемые файлы для Git
│   └── .env.example                 # Пример настроек окружения
│
├── 🐍 Backend - Telegram Bot (Python + aiogram)
│   └── bot/
│       ├── main.py                  # Точка входа бота
│       ├── requirements.txt         # Python зависимости
│       └── middlewares/
│           └── phone_verification.py # Проверка номера + блокировка РФ/BY/IR/IQ
│
├── 🔧 Backend - REST API (Python + FastAPI)
│   └── backend/
│       └── api/
│           ├── swipe_handler.py     # Обработка свайпов, мэтчей
│           └── filters.py           # API настроек фильтров
│
├── ⚛️ Frontend - Web App (React + Tailwind)
│   └── frontend/
│       ├── package.json             # NPM зависимости
│       ├── tailwind.config.js       # Настройки Glassmorphism дизайна
│       ├── filter-settings-demo.html # Демо фильтров (standalone)
│       └── src/
│           ├── App.jsx              # Главный компонент
│           ├── components/
│           │   ├── Discovery.jsx    # Экран свайпов (Tinder-like)
│           │   └── FilterSettings.jsx # Экран настроек
│           └── utils/
│               └── telegram.js      # Wrapper для Telegram WebApp API
│
└── 🗄️ Database (PostgreSQL + PostGIS)
    └── database/
        └── schema.sql               # Полная схема БД с триггерами

🎯 Ключевые файлы для изучения:

1. bot/middlewares/phone_verification.py
   → Логика блокировки регионов по номеру телефона

2. frontend/src/components/Discovery.jsx
   → UI свайпов с Glassmorphism эффектами

3. database/schema.sql
   → Структура БД + автоматические триггеры мэтчей

4. backend/api/swipe_handler.py
   → Обработка свайпов и создание мэтчей

5. premium-dating-app.html
   → Полностью рабочее демо приложения


🚀 Быстрый запуск:

Вариант 1 (Демо без установки):
  1. Откройте premium-dating-app.html в браузере
  2. Готово! Играйтесь со свайпами

Вариант 2 (Полная установка):
  1. Читайте БЫСТРЫЙ_СТАРТ.md
  2. Установите PostgreSQL
  3. Запустите бота (bot/main.py)
  4. Запустите фронтенд (npm run dev)


📦 Технологии:

Backend:
  • Python 3.11+
  • aiogram 3.x (Telegram Bot)
  • FastAPI (REST API)
  • PostgreSQL 14+ (База данных)
  • PostGIS (Геолокация)

Frontend:
  • React 18
  • Vite (Сборка)
  • Tailwind CSS 3
  • Framer Motion (Анимации)
  • Telegram WebApp SDK


🎨 Дизайн:
  • Стиль: Glassmorphism (матовое стекло)
  • Цвета: Электро-индиго (#6366F1), Фиолетовый
  • Тема: Тёмная с градиентами
  • Шрифты: System UI fonts
```
